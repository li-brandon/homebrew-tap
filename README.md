# Homebrew Tap for lazyliner

This is the official Homebrew tap for [lazyliner](https://github.com/li-brandon/lazyliner), a beautiful, keyboard-driven terminal TUI for Linear.

## Installation

```bash
brew tap li-brandon/tap
brew install lazyliner
```

## Upgrading

```bash
brew update
brew upgrade lazyliner
```

## About

The formula in this tap is automatically maintained by [GoReleaser](https://goreleaser.com/) on each release of lazyliner. The formula supports both Intel and Apple Silicon Macs.

For more information, visit the [main project repository](https://github.com/li-brandon/lazyliner).
